# Report

LaTeX report source files for the final project submission.

## Files

- `main.tex`: Main report document.
- `macros.tex`: Shared commands and formatting helpers.
- `references.bib`: Bibliography database.
- `figures/`: Report figures and plots used in the document.

## Overleaf Workflow

Use Overleaf as the primary editing and compile workflow.

1. Create a new project in Overleaf.
2. Upload the full contents of this `report/` folder:
	- `main.tex`
	- `macros.tex`
	- `references.bib`
	- `figures/` directory
3. Set `main.tex` as the main document if Overleaf does not auto-detect it.
4. Recompile and check for missing figure/citation paths.

## Export

- Use Overleaf's PDF download for final submission artifacts.
- If you need source export, use Overleaf's "Download as ZIP" option.

## Editing Notes

- Keep figure filenames stable once referenced in `main.tex`.
- Update citations in `references.bib` before final export.
